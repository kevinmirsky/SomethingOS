\section{Problem One}

\textit{How is your	console	like the ancient TTY subsystem in Unix as described in 
https://www.linusakesson.net/programming/tty/ ?}

Both the console we've implemented and the TTY subsystem use a buffer to store the text the user enters in the line. This is what lets the user hold onto what they're writing before sending it to the program. It also only does something when called upon by a process or the kernel. For example, our text is only processed because of the device driver calling the method for our console to handle input.
